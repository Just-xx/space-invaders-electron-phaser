# Space Invaders Clone

A simple implementation of the classic Space Invaders game created using the Phaser 3 framework and Electron.

## Description

In this game, the player controls a spaceship at the bottom of the screen, tasked with destroying incoming waves of aliens. The aliens move as a group to the left and right, and after reaching the edge of the screen, they move down and speed up.

## Controls

-   **Left Arrow**: Moving the ship to the left.
-   **Right Arrow**: Moving the ship to the right.
-   **Space**: Shooting
-   **Escape**: Pause
